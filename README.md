# AI Day 3 - EDA & Statistical Thinking Challenge

## Project Overview
This repository contains the Day 3 Exploratory Data Analysis (EDA) activity for the AI Internship program.

## Dataset
E-Commerce Customer Dataset with features:
- Customer Age
- Income
- Purchase Amount
- Number of Purchases
- Customer Lifetime Value
- Purchase Category (High/Medium/Low)

## Files
- `data/ecommerce_customers.csv` - Raw dataset
- `notebooks/EDA_Ecommerce_Analysis.ipynb` - EDA notebook
- `docs/management_summary.md` - Management summary

## Analysis Parts
1. Descriptive Statistics
2. Data Distribution Analysis
3. Sampling & Data Splitting Strategy
4. Correlation Analysis
5. Outlier Detection
6. Target Variable Analysis
7. Leakage Detection
8. Visualization & Business Findings
